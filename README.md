# 👑 Golden Sands

> Where luxury meets the absolute pinnacle of wine crafting. No cap.

Welcome to the **Golden Sands** repository. This isn't just a winery website template; it's an absolute masterpiece of digital elegance, built for those who understand true quality. We're talking high-end animations, dark-mode-first design, and gold accents that scream "premium."

## ✨ The Flex (Features)

- **Preloader:** Smooth, buttery animated brand reveal.
- **Hero Section:** Ken Burns background effects that make you feel like you're standing in the vineyards. Staggered reveals. Count-up stats that speak for themselves.
- **Wine Showcase:** A tabbed, glowing shrine to our finest bottles.
- **Winery Carousel:** Auto-advancing slides that take you on a visual tour of Golden Sands.
- **Museum & News:** Dive deep into the history and updates of the estate.
- **Flawless Animations:** Scroll-triggered fade-ups, slide-ins, and scale-ins that just *hit different*.
- **Responsive:** Looks 10/10 on any device.

## 🛠️ The Tech Stack (Built Different)

- **React 18 + TypeScript:** Because we like our code strictly typed and lightning fast.
- **Vite:** Blazing fast builds.
- **Tailwind CSS:** For that pixel-perfect, utility-first styling.
- **Lucide React & shadcn/ui:** Top-tier icons and components.

## 🚀 Quick Start (Get It Running)

Ready to see greatness? Let's go.

```bash
# Install the dependencies
npm install

# Spin up the dev server
npm run dev
```

## ⚙️ Configuration (Your Canvas)

Everything you need to customize is sitting right in `src/config.ts`. It's literally a single source of truth. Fill it out, and watch the site transform.

## 🎨 Design Vibes

- **Gold Palette:** `#d2a855`, `#dbb977` – Only the finest gold tones.
- **Dark Mode Palette:** `#141414`, `#0e0e0e` – Deep, rich dark backgrounds.
- **Typography:** *Cormorant Garamond* (headings), *Poppins* (body), and *Qwitcher Grypen* (for that elegant script drip).

## 📝 Notes for the Devs

- The site uses Formspree for the contact and newsletter forms. Easy to set up, zero backend required.
- Make sure to drop your optimized images into `public/images/`.
